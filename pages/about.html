About Spellbuild

Spellbuild is a Shooter base game. This game use survival mode, so you have to survive from 100 players around the world

Features

You are not using a weapon. Instead, you use melee weapons to survive!

Basic Mechanics

  1. W = Walk Forward
  2. S = Walk Backward
  3. A = Walk Left
  4. D = Walk Right
